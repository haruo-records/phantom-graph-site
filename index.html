Phantom Graph

A perceptual phenomenon where motion emerges between still images.

We do not only see what is there.
We also see what is about to happen.

When something is slightly incomplete or separated, our mind quietly connects it — and in that moment, we begin to feel motion, change, and life.

Definition

Phantom Graph is a visual format where motion emerges from the viewer's unconscious interpolation between still images.

It is not animation.
The motion does not exist in the images themselves — it is constructed by the observer.

What is Phantom Graph

Phantom Graph is a visual format in which motion emerges not from actual animation, but from the viewer's unconscious interpolation between multiple still images.

When two or more images are placed side by side with intentional gaps, the human brain fills in the missing continuity. This creates a perceived motion that does not physically exist.

Phantom Graph is not an animation. It is a structure that reveals how perception itself generates movement.

Core Principle

Motion is not always displayed.
It can be constructed inside the observer.

The images themselves do not move.
The observer creates the motion.

Format

A Phantom Graph typically:

  • Uses multiple still images
  • Places them side by side with intentional spacing
  • Introduces slight differences between each image
  • Suggests motion without explicitly rendering it

If motion exists without interpretation, it is not Phantom Graph.

What Phantom Graph is NOT

  • GIF animations
  • Video loops
  • Frame-by-frame motion playback
  • Explicit transitions

If motion exists without the viewer's interpretation, it is not Phantom Graph.

Why it matters

We do not passively receive images.
We actively construct reality.

Phantom Graph reveals this hidden process.

Usage

Phantom Graph can be applied to visual art, design, installations, social media formats, and Slow Watching. It is both a format and a way of seeing.
